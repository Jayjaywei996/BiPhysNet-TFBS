# Security and Privacy

Do not upload private or sensitive materials to this repository.

Avoid committing:

- unpublished manuscript drafts;
- patent drafts or disclosure documents;
- private datasets;
- API keys or tokens;
- model checkpoints;
- pretrained model weights;
- local absolute paths containing personal information.

If you find a security or privacy issue, open a GitHub issue or contact the repository owner through an appropriate private channel.
