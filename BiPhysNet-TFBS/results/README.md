# Results

This directory stores lightweight, GitHub-friendly experiment summaries.

Recommended files to commit:

- benchmark summaries;
- small evaluation tables;
- selected figures;
- experiment notes;
- exact command records.

Do **not** commit:

- raw datasets;
- `.npy` feature matrices;
- model checkpoints;
- full logs;
- TensorBoard folders;
- W&B runs.

The current benchmark note is available in [`benchmark_summary.md`](benchmark_summary.md).
